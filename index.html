AKAN NAMES PROJECT

Descripition of the Project

The project is a web-application is that generates a name of a Ghananian child based on the date he/she was born and their gender. It provides a form where an individual or the user should input their birthdate and choose their gender. Dates should follow a particular format as will be explained in the subsequent paragraphs. Non adherence to the date format will result to a snap. The project has adopted several technologies to ease the workability of the application.
