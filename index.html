AKAN NAMES PROJECT

Descripition of the Project

The is a web application which generates a name of a Ghananian child based on their gender and birthdate.The user should input their birthdate and choose their gender. The project has adopted several technologies like HTML, CSS, JS among others to ease the working of the application.


Ghananian Names and their corresponding days of the week

Here are the Male Akan Names

  • Kwasi-Sunday
  • Kwadwo-Monday
  • Kwabena-Tuesday
  • Kwaku-Wednesday
  • Yaw-Thursday
  • Kofi-Friday
  • Kwame-Saturday

Here are the Female Akan Names

  1. Akosua-Sunday
  2. Adwoa-Monday
  3. Abenaa-Tuesday
  4. Akua-Wednesday
  5. Yaa-Thursday
  6. Afua-Friday
  7. Ama-Saturday
